# Architect OS — AZ-305 Coach

> Ein modernes, kostenloses Lernsystem für die Microsoft-Zertifizierung **AZ-305: Designing Microsoft Azure Infrastructure Solutions**.
> Gebaut von **Tarek** · Cloud Engineer ☁️

Jederzeit starten, im eigenen Tempo lernen — die App merkt sich deinen Fortschritt automatisch im Browser. Kein Login, kein Server, keine Installation.

## ✨ Features

- **14-Tage-Sprint** — strukturierter Lernplan zum Abhaken, datumslos
- **150 Architektur-Cases** über 13 Domains — eigene Lernfragen, keine Exam-Dumps
- **Spaced Repetition** — falsch beantwortete Fragen kommen gezielt zurück
- **Exam-Simulation** — 50 Fragen, 120 Minuten, echte Prüfungsbedingungen
- **8 Design-Labs** mit Lösungsskizzen (Landing Zone, Netzwerk, BCDR, Identity, Security, Cost, Data, Migration)
- **32 Decision Cards** — Service-Abgrenzungen auf einen Blick
- **PDF-Spickzettel** — 1 Seite pro Domain zum Ausdrucken
- **Analytics** — schwache Domains, Score-Verlauf, gemeisterte Fragen
- **Dark Mode** · **Export/Import** des Fortschritts als JSON

## 🚀 Nutzen

Online: einfach den Link öffnen.
Lokal: `index.html` im Browser öffnen — fertig. Voll offline-fähig, keine externen Abhängigkeiten.

## 🛠 Tech

Vanilla **HTML / CSS / JavaScript**, Fortschritt via `localStorage`. Kein Build, kein Backend.

## ⚖️ Hinweis

Inoffizielles Lernhilfsmittel. Ausschließlich selbst formulierte Lernfälle und Microsoft Learn als Referenz — **keine** echten Prüfungsfragen oder Exam-Dumps.
